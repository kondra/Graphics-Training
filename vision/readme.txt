Программа имеет консольный интерфейс. (Запускать необходимо из консоли, например из FAR)
Опция --help выдает справку по использованию программы. Bootstrapping при обучении сделан опциональным.
Перед подсчетом HOG применялся (сепарабельный) фильтр Гаусса с sigma=1.
Для увеличиния скорости работы HOG подсчитывается сразу для всей картинки.
При подсчете HOG брались x-1 и x+1 пиксели.
Так же размер HOG гистограммы был выбран равным 16 а не 8 (показал лучшие
результаты).
Обучение проводилось под Linux. Под Windows результат был другой.
(видимо liblinear ведет себя по разному)
liblinear добавлен в проект и статически линкуется с ним.
